# @amboss/design-system docs

These docs were bundled into the CLI at publish time and copied locally during scaffolding. They do not update automatically.

To refresh with the latest DS docs:

```bash
curl -o docs/amboss/llms.txt https://design-system.miamed.de/llms.txt
curl -o docs/amboss/llms-full.txt https://design-system.miamed.de/llms-full.txt
```

Or update create-amboss-app to a newer version which bundles fresher docs.
